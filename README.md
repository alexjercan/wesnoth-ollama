# Wesnoth Ollama Modification

Very simple modification that enables the usage of ollama in BfW.

This kind of requires my fork of Wesnoth to work because it needs the extra
`[ollama]` tag.

IT REQUIRES <https://github.com/alexjercan/wesnoth/tree/feature/genai-1>

### Quickstart

Copy this to the `add-on` folder and play the game using my build :)
